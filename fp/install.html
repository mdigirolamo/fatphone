FatPhone Installer

Before running this installer, please:

  • Create a site called FatPhone (name mandatory)
  • Create a FlexFamily called FatPhone with the following type names (mandatory):

    Type Name Flex Component
    fp_A Attributes
    fp_CD Content Definition
    fp_PD Parent Definition
    fp_P Parent
    fp_C Content
IMPORTANT!!! On Windows I experienced a long delay when posting assets.
Please be patient when the installer looks to have stalled (it actually is not).
